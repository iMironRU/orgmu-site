Как открывать. Страницы связаны между собой и подгружают общие блоки (шапка, меню, подвал) по сети — поэтому проект должен быть развёрнут на HTTP-хостинге (Netlify, GitHub Pages, любой веб-сервер). При открытии файлов напрямую двойным кликом (file://) общие блоки не загрузятся.

Импорт в Figma. Разверните сайт, затем в плагине html.to.design укажите URL нужной страницы — он отрисует её как фрейм. Список страниц ниже — это и есть каталог адресов для импорта.
